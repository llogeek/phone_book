Класс Interface используется для организации взаимодействия пользователя с классом PhoneBook. 
В данном классе реализуются следующие методы:
     init() - инициализация обьекта класса. Cоздается обьект класса PhoneBook, который является атрибутом класса Interface. 
              Можноо указать название файла для вывода информации и максимальное количество запиcей на странице.
     welcome() - функция выводит на экран функционал программы для пользователя.
     print_data() - выводит весь список 
     process_menu() - функция принимает в качестве аргумента запрос пользователя (результат выполнения функции welcome()) и вызывает соответствующий запросу функционал.
     add_data() - функция запрашивает у пользователя данные для внесения в телефонную книгу.
                  Производится проверка ФИО на пустоту и наличие такой же записи в таблице.
                  Форматы допустимых для ввода номеров телефонов:
                   8 (3 цифры) (7-10 цифр пробелов или -),
                   8-(3 цифры)-(7-10 цифр пробелов или -),
                   7 (3 цифры) (7-10 цифр пробелов или -),
                   7-(3 цифры)-(7-10 цифр пробелов или -).
     edit() - редактирование записей в телефонной книгею+.
              Функция запрашивает ФИО и название организации для поиска в данных.
              Также пользователь имеет возможность редактировать несколько полей.
              Порядковы номера редактируемых данных вводятся в одну строку c/без пробела.
              В случае, если данных в таблице нет, выводится соответствующее предупреждение.
     search() - функия поиска записей в книге по нескольки полям.
                Порядковые номера полей вводятся в одну строку с/без пробела.
                Возможен поиск по нескольким полям. 
                Если данные не найдены,выводится соответствующее предупреждение.

Класс PhoneBook реализует телефонную книгу.
Атрибутами класса являются: максимально количество записей на странице, имя файла для записи и DataFrame для харнения данных.
В данном классе реализуются следующие методы:
     init() - создание обьекта класса телефонная книга. Аргументы: название файла для записи данных (по умолчанию  phone_book.csv), максимальное количество записей на странице (по умолчанию 10).
     str() - преобразование данных класса в строковый обьект (используется методом print_data() из класса Interface).
     insert() - функция добавления данных в DataFrame.Используется методом add_data() класса Interface.
                В качестве данных передается набор именованных аргументов.
     add_to_csv() - функция добавляет данные, добавленные пользователем, в конец файла.
     write_to_csv() - перезапись данных в файл csv. Используется для записи данных после изменения.
     get_data() - получения данных DataFrame.
     get_field_name() - функция по номеру поля данных возвращает его название.
     search() - поиск данных в заданном DataFrame (передается как аргумент) по номеру поля (название поля в функции получается с помощью функции get_field_name()) и значению.
     search_all() - поиск по значениям полей Фамилия, Имя, Отчество и Организация. Используется функцией add_data() класса Interface.
     edit() - редактироание записей. Используется функцией edit() класса Interface.
